## ✨ 關於這個網站

這是 **YuChiao Chou（周語喬）** 的個人禪繞畫藝術空間。

十年前，一場偶然的相遇讓她走進了 Zentangle 的世界。透過一支筆、一張紙、重複的筆觸與圖樣，她在方寸之間找到了內在的平靜。如今，她希望將這份禮物傳遞給更多人——讓每一個願意拿起筆的人，都能在創作的過程中遇見自己。

## 🎨 設計風格關鍵詞

| 元素 | 設計語言 |
|------|----------|
| **色彩** | 米白、暖灰、墨黑、淡金、霧藍、藕粉 |
| **字體** | 中文：思源宋體 / 英文：Playfair Display + Lato |
| **質感** | 水彩渲染、紙張紋理、手繪筆觸、留白呼吸 |
| **動效** | 緩慢浮動、筆觸繪製動畫、墨暈擴散、花瓣飄落 |
| **氛圍** | 寧靜、優雅、夢幻、療癒、專注、當下 |
