## 3. 頁面設計概念

### 🏠 首頁（Hero Experience）
- 全螢幕沉浸式開場：手繪筆觸逐漸浮現成完整圖樣
- 浮動標語：*"In every stroke, there is a meditation."*
- 四個入口區塊：Gallery / Learn / About / Shop

### 👤 關於頁（Storytelling）
- 時間軸呈現 10 年旅程
- 手寫體引述與照片交錯
- 核心理念：*"Zentangle is not about the result. It is about the process."*

### 🖼️ 畫廊（Visual Journey）
- 瀑布流或 Masonry 排版
- 點擊放大 + 作品故事
- 分類：原創圖樣 / 彩色創作 / 卡片設計 / 大型作品

### 📚 學習頁（Resource Hub）
- 初學者三步驟：邊框 → 圖樣 → 陰影
- 可下載圖樣練習紙（PDF）
- 推薦工具與材料

### 🛒 商店（Curated Selection）
- 原作畫作（限量）
- 手繪卡片組
- 線上課程 / 工作坊預約

---

## 4. 互動細節

- **游標**: 自定義細筆游標，hover 時有墨暈效果
- **載入**: 筆觸繪製進度條
- **滾動**: 視差效果，元素如花瓣般緩緩浮現
- **畫廊**: hover 時顯示作品名稱與創作日期
- **表單**: 輸入框有手寫邊框動畫
