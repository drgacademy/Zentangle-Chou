# 04 — 經典圖樣字典

## 引言：禪繞圖樣的命名文化

「Tangle」（圖樣）是禪繞畫的基本單元——一個由少數重複筆劃所組成的結構化圖樣，可以填進紙磚的任何區塊。禪繞畫被稱為「解構式圖樣畫」（deconstructed pattern drawing），意思是不論一個圖樣看起來多複雜，都能拆解成 2–6 種最基本的筆劃：直線、點、圓、彎弧、弧形、S 曲線。

命名文化是禪繞迷人的一部分。創辦人 Rick Roberts 與 Maria Thomas 為許多原創圖樣命名時，會以家人、旅行的地點、或畫起來的「手感」為靈感——就像台灣老師傅替自家茶取名「東方美人」「鐵觀音」一樣，名字裡藏著故事。**Hollibaugh** 取自 Maria 的女兒 Molly Hollibaugh（她現在也是禪繞認證教師，並擔任公司首席設計師）；**Chillon** 取自瑞士的中世紀古堡西庸城；**Zinger** 取自它「彈跳」的能量感；**Mooka** 則是 Maria 看到捷克新藝術畫家 Mucha 的石版海報而得到的靈感。

當其他禪繞認證教師（CZT）創作新圖樣，並透過 Linda Farmer 的網站 **TanglePatterns.com** 發表時，命名也通常延續這份精神：簡短、有意境、常常帶點俏皮。沒有任何中央機構在審核「官方圖樣」——這真的是一場民間的傳承。一個圖樣只要有人畫、有人傳，就成了禪繞詞彙的一份子。

下文是超過四十個經典圖樣的工作字典。每一條包含：英文名與中文譯名（沿用既有翻譯，無既譯則標「暫譯」）、創作者、文字繪製步驟、以及最早出現的書／資源。**所有描述只用文字呈現步驟**，內容改寫自 Linda Farmer 的線上典藏與多本禪繞官方書籍，不複製任何圖像。

---

## 字典如何分類

依主要創作者分為四節：

- **A 組** — 官方禪繞圖樣，由 Rick Roberts、Maria Thomas，以及 Roberts/Thomas 家族成員（Molly Hollibaugh、Martha Huggins）創作。
- **B 組** — Sandy Steen Bartholomew（Beez in the Belfry 工作室）創作的圖樣。
- **C 組** — Linda Farmer（TanglePatterns.com 創辦人）創作的圖樣。
- **D 組** — 其他禪繞認證教師（CZT）與重要貢獻者的圖樣。

每節內，圖樣大致依學習者最早接觸到的順序排列。

## A 組 — 官方禪繞圖樣（Roberts／Thomas 家族）

### 1. Crescent Moon — 新月紋 / 弦月紋

- **創作者：** Rick Roberts & Maria Thomas
- **首見：** 禪繞最初教材；CZT 認證課的「第一個圖樣」。
- **畫法：**
  1. 沿著區塊邊緣畫一排朝內的半月形（小弧凸起）。
  2. 把每個半月填滿黑色。
  3. 在每個黑色凸起外側畫一條跟著弧度走的細線——這是第一道「光環」（aura）。
  4. 繼續往外畫第二道、第三道光環，直到填滿區塊。
- **為什麼重要：** Crescent Moon 教會「光環」這個基礎概念——一條線跟著前一個形狀的輪廓向外回應——這個技法之後會出現在數十個其他圖樣裡。

### 2. Hollibaugh — 哈利堡紋 / 木條交錯紋

- **創作者：** Maria Thomas（取自女兒 Molly Hollibaugh 之名）
- **首見：** 早期禪繞教材；《Zentangle Primer Vol. 1》。
- **畫法：**
  1. 畫一條長帶（兩條平行曲線、兩端為圓形），橫越區塊。
  2. 再畫一條帶子，看起來像「從第一條後面穿過去」——當線觸到第一條時，提筆，到另一邊再繼續。
  3. 持續加入帶子，每一條都從前面那條的後方穿過。
  4. 條與條之間的縫隙填黑。
- **為什麼重要：** Hollibaugh 教會「畫在後面」這個技法——是製造「立體錯覺」最可靠的方式，就像竹編籃子的層次感。

### 3. Printemps — 春天紋 / 蝸旋紋

- **創作者：** Maria Thomas
- **首見：** 原始禪繞牌組；《One Zentangle a Day》第 1 天。
- **畫法：**
  1. 從一個小點或緊密的小捲開始。
  2. 用連續的曲線往外旋轉，每一圈間距盡量一致。
  3. 結尾以一個小勾或小弧收住。
  4. 把許多旋紋緊密排列，留下小三角形空隙。
- **為什麼重要：** Printemps 是所有圖樣中最讓人冷靜的——重複的旋轉節奏像是手心搓念珠，極具冥想效果。

### 4. Cadent — 對偶迴環紋

- **創作者：** Rick Roberts & Maria Thomas
- **首見：**《Zentangle Primer Vol. 1》。
- **畫法：**
  1. 在區塊內輕點一格網狀的點。
  2. 用「向外凸的弧線」連接每對相鄰的點——先全部水平方向、再全部垂直方向。
  3. 結果形成一張「凹邊菱形」的網絡。
  4. 在每個菱形中畫一個小圓、一個點，或填一塊黑心。
- **為什麼重要：** Cadent 示範了「隱藏的網格」如何帶來秩序而不顯得僵硬。

### 5. Mooka — 慕卡藤蔓紋

- **創作者：** Maria Thomas（靈感來自捷克新藝術畫家 Alphonse Mucha 的石版畫）
- **首見：**《Zentangle Primer Vol. 1》；Maria 紙磚示範常見。
- **畫法：**
  1. 從一條柔順的長 S 曲線開始，作為主莖。
  2. 從莖上延伸出水滴形的瓣，往莖的方向回捲。
  3. 每片瓣以連續的「套索」筆法畫成——從莖出發、繞回莖。
  4. 在主瓣裡再加更小的次級瓣，營造層次。
- **為什麼重要：** Mooka 是禪繞中的「有機代表」——與規則網格圖樣完全相反的流動感，像水墨畫裡的牡丹枝條。

### 6. Florz — 地磚紋

- **創作者：** Rick Roberts & Maria Thomas
- **首見：** 早期禪繞牌組；《One Zentangle a Day》。
- **畫法：**
  1. 鋪一格網狀的點。
  2. 把每個點對角連接，形成「立著的小方塊」（菱形）格網。
  3. 四個菱形角相交處，放一個小黑菱或填黑點。
  4. 視需要在每個大菱形中心點一個小點。
- **為什麼重要：** Florz 是最簡單的網格圖樣，是進入「重複韻律」世界的第一步——像台北老房子的磨石子地磚。

### 7. 'Nzeppel — 義式麵餃紋

- **創作者：** Rick Roberts & Maria Thomas（命名來自一種義大利麵餃）
- **首見：**《Zentangle Primer Vol. 1》。
- **畫法：**
  1. 鬆鬆地畫一些不太規則的水平與垂直線條，分成許多格子。
  2. 在每個格子裡，畫一個飽滿的「枕頭形」——幾乎、但不完全頂到四個邊。
  3. 枕頭間的細縫形成柔軟的鵝卵石網格。
  4. 視需要在枕頭某一側做陰影。
- **為什麼重要：** 像一籠剛蒸好的小餃子——枕頭給 'Nzeppel 一種無需明顯陰影就能感覺到的觸覺與層次。

### 8. Tipple — 圓珠紋

- **創作者：** Rick Roberts & Maria Thomas
- **首見：** 早期禪繞牌組。
- **畫法：**
  1. 用大小不一的圓填滿區塊。
  2. 先放最大的圓，讓中等的填補空隙，最小的填空隙的空隙。
  3. 視需要在每個圓的某一側加陰影，讓它看起來像泡泡。
- **為什麼重要：** Tipple 是萬用「填料」——當區塊需要溫和質地、又不想搶走主圖樣時就用它，像茶碗下的小竹墊。

### 9. Ahh — 啊光紋 / 星芒紋

- **創作者：** Rick Roberts & Maria Thomas
- **首見：** 早期禪繞牌組。
- **畫法：**
  1. 畫一個小「Y」字，或三線星形。
  2. 加上第四、第五條光線，形成小型放射星芒。
  3. 把這種星芒散布在區塊各處，大小變化。
  4. 在星芒之間點上小點，作為呼吸空間。
- **為什麼重要：** Ahh 像是紙上的小聲嘆息——名字「啊」就是放鬆的吐氣聲。

### 10. Bales — 包袱紋 / 蓮葉紋

- **創作者：** Rick Roberts & Maria Thomas
- **首見：** 原始禪繞牌組。
- **畫法：**
  1. 鋪一格方格網。
  2. 每個方格內，從相對的兩個角向外畫兩條外凸的弧——它們在中央相遇，形成一個四瓣枕頭。
  3. 相鄰方格的弧自然連接，整體像鋪滿柔軟枕頭的綢被。
  4. 在花瓣中心加點或陰影。
- **為什麼重要：** Bales 是最溫柔的網格圖樣之一；曲線把網格的剛硬感全洗掉了。

### 11. Static — 靜電紋 / 鋸齒交錯紋

- **創作者：** Rick Roberts & Maria Thomas
- **首見：** 早期禪繞牌組。
- **畫法：**
  1. 畫一條垂直線。
  2. 從線上射出一排鋸齒狀的閃電。
  3. 再畫第二條垂直線；從它射出的鋸齒朝向第一組，像齒輪互咬。
  4. 持續以交替欄位填滿；視需要把內三角填黑。
- **為什麼重要：** Static 帶來高對比、能量感的圖樣——適合製造視覺衝擊。

### 12. Onamato — 鳴音紋 / 聲響紋

- **創作者：** Maria Thomas（取自 onomatopoeia「擬聲詞」之諧音）
- **首見：**《Zentangle Primer Vol. 1》。
- **畫法：**
  1. 沿基線畫一排相鄰的圓。
  2. 上方再畫一排相鄰的圓，與下排錯開半個圓。
  3. 在每個圓的內側畫一道細光環。
  4. 三個圓交會的小縫處填黑。
- **為什麼重要：** Onamato 示範了「光環往內畫」（aura inside）也能創造立體感。

### 13. Knightsbridge — 棋盤格紋

- **創作者：** Rick Roberts & Maria Thomas
- **首見：** 原始禪繞牌組。
- **畫法：**
  1. 畫一格方格網。
  2. 把交錯的方格填黑，像棋盤。
  3. 視需要把方格旋轉 45° 形成菱形棋盤。
- **為什麼重要：** Knightsbridge 是教科書級的「定錨圖樣」——一塊純黑白對比區，會讓旁邊的圖樣看起來更精緻。

### 14. Sez — 切疊紋

- **創作者：** Maria Thomas
- **首見：**《Zentangle Primer Vol. 1》。
- **畫法：**
  1. 沿基線畫一排朝上的三角形。
  2. 上方再畫一排朝上的三角形，位於下排的兩個尖點之間。
  3. 持續往上堆疊，形成三角的鑲嵌結構。
  4. 把交替的三角形填黑。
- **為什麼重要：** Sez 是最快創造「織紋／鱗片」錯覺的方法之一。

### 15. Pokeleaf（Pokeroot） — 戳葉紋 / 小球葉紋

- **創作者：** Maria Thomas
- **首見：**《Zentangle Primer Vol. 1》。
- **畫法：**
  1. 畫一條彎彎的莖。
  2. 在莖上長出圓鼓鼓的氣球狀葉。
  3. 每片葉與莖相接處有一個小凹口。
  4. 加細葉脈或單側陰影。
- **為什麼重要：** Pokeleaf 是經典的「藤蔓／圓葉」有機圖樣，常用於邊框。

### 16. Beelight — 蜂光紋 / 蜂巢光紋

- **創作者：** Sandy Steen Bartholomew（因普及度，常被歸入官方圖樣）
- **首見：**《Yoga for Your Brain》。
- **畫法：**
  1. 畫一個六角或圓潤的蜂巢格。
  2. 格內畫一個彎曲的「火焰」形。
  3. 上面或旁邊再畫一格，火焰朝向前一格。
  4. 周圍填黑或畫斜線。
- **為什麼重要：** Beelight 是「植物與幾何」之間的橋樑。

### 17. Fescu — 細草紋 / 草尖紋

- **創作者：** Maria Thomas
- **首見：** 早期禪繞牌組。
- **畫法：**
  1. 從基線畫一條細的 J 字曲線，尾端加一個實心圓點。
  2. 旁邊再畫一條 J 字曲線，方向類似但角度略不同。
  3. 沿區塊持續畫，整體像隨風起伏的草。
  4. 尖端的小點是它的特徵。
- **為什麼重要：** Fescu 適合做邊界或填料，與較重的鄰居形成對比。

### 18. Shattuck — 沙塔克紋 / 弧網紋

- **創作者：** Maria Thomas（地名命名）
- **首見：**《Zentangle Primer Vol. 1》。
- **畫法：**
  1. 在區塊上畫一條波浪狀水平線。
  2. 從波浪向下畫一連串如「裙擺」般的垂直弧。
  3. 在弧下方再畫一條波浪線，與第一條相呼應。
  4. 持續往下堆疊；弧的內部填黑。
- **為什麼重要：** Shattuck 提供強烈的水平節奏——視覺像一排懸吊的黑色果實。

### 19. Zinger — 彈簧紋 / 飛鏢紋

- **創作者：** Maria Thomas
- **首見：**《Zentangle Primer Vol. 1》。
- **畫法：**
  1. 畫一條像逗號的彎莖。
  2. 一端加上一個小旋或小捲。
  3. 在莖上加一些平行斜線或小葉。
  4. 在區塊內隨意散布許多 Zinger，方向各異。
- **為什麼重要：** Zinger 嚴格說是「裝飾元素」——是錦上添花的小花俏。

### 20. Paradox — 悖論紋 / 旋角紋

- **創作者：** Rick Roberts
- **首見：** 原始禪繞教材；Rick 經常示範。
- **畫法：**
  1. 從一個三角形（或任何近正多邊形）開始。
  2. 從每個頂點向相鄰邊上的某一點畫直線——但不畫到底，留一段距離繼續往內走。
  3. 每一條新線在內部創造一個更小的三角形；持續向中心螺旋。
  4. 由純直線製造出「螺旋漩渦」的視覺幻覺。
- **為什麼重要：** Paradox 是最具視覺衝擊的圖樣之一——「直線製造曲線錯覺」的經典範例。

### 21. Munchin — 棕櫚編織紋

- **創作者：** Maria Thomas
- **首見：**《Zentangle Primer Vol. 1》。
- **畫法：**
  1. 畫兩條平行曲線形成一條帶。
  2. 帶內畫連續的對角編織——粗細交替的緞帶。
  3. 緞帶交織如竹編。
  4. 在「下層」緞帶加深陰影。
- **為什麼重要：** Munchin 帶來強烈的「編織」質感，常被填入 Hollibaugh 式的長帶內。

### 22. Flux — 流葉紋

- **創作者：** Rick Roberts & Maria Thomas（兩人各有版本）
- **首見：**《Zentangle Primer Vol. 1》及多個 CZT 課程。
- **畫法：**
  1. 畫一片像火焰般彎曲的葉。
  2. 沿葉長軸加一條彎曲的葉脈。
  3. 在莖上聚集多片這樣的葉，每片捲度不一。
  4. 大小有變化；讓部分葉「藏在」其他葉後面。
- **為什麼重要：** Flux 是「葉、羽、流動邊」的工作主力。共有 Maria 版與 Rick 版，葉形略有差異。

### 23. Chillon — 西庸城堡紋

- **創作者：** Maria Thomas（取自瑞士西庸城堡）
- **首見：** 原始禪繞牌組。
- **畫法：**
  1. 鋪一格方格網。
  2. 每格內畫一個粗臂十字，中央留一小方塊。
  3. 把相鄰方格的十字臂連接起來，形成連續的「希臘回紋」。
  4. 把空格填黑。
- **為什麼重要：** Chillon 把簡單網格變成迷宮般、像古建築外牆的圖樣。

### 24. Diva Dance — 女神之舞 / 舞動紋

- **創作者：** Maria Thomas（Molly Hollibaugh 廣為教學）
- **首見：** 禪繞官方教學影片與「Diva Dance」課程。
- **畫法：**
  1. 畫一條長且不對稱的 S 形——一端粗鈍、另一端細捲。
  2. 旁邊再畫一條 S 形，靠著前一條鋪。
  3. 不斷堆疊；每一條都呼應、推移前一條。
  4. 整體像隨風飄動的長葉，或一團跳動的火苗。
- **為什麼重要：** Diva Dance 是「連續流動」的冥想——廣受喜愛，最適合填大塊空白。

### 25. Bunzo — 邦佐紋 / 月珠紋

- **創作者：** Sandy Steen Bartholomew
- **首見：**《Yoga for Your Brain》。
- **畫法：**
  1. 畫一個圓潤的形（略扁的球）。
  2. 內部畫一連串平行的半月弧，全部偏向同一邊。
  3. 看起來像一顆滾動、分節的珍珠。
  4. 沿莖串連許多 Bunzo。
- **為什麼重要：** 巢套的弧線讓整串看起來像一條珠鍊。

### 26. Cubine — 立方體紋

- **創作者：** Maria Thomas
- **首見：** 早期禪繞牌組。
- **畫法：**
  1. 畫一格方格網。
  2. 每格內從四個角畫向中心的對角線，但不到中心——留下一個小方塊。
  3. 把對角的兩個三角填黑，另兩個留白。
  4. 整片讀起來像 3D 立方體的鑲嵌。
- **為什麼重要：** Cubine 是最簡單就能呈現「立體錯覺」的圖樣。

### 27. Drawings — 描畫紋（即「string」概念）

- **創作者：** Rick Roberts & Maria Thomas
- **首見：** 禪繞官方方法；《Zentangle Primer Vol. 1》。
- **畫法：** 這是「鉛筆 string」的官方概念，用以將紙磚分區。輕輕用鉛筆在紙磚上畫一條自由曲線——可以是簡單的 Z、也可以是蜿蜒的塗鴉——把方形分成 3–6 個區塊。每個區塊就是一個圖樣的畫布。
- **為什麼重要：** String 是禪繞的靈魂：它建立結構，卻不規定結果。

## B 組 — Sandy Steen Bartholomew 圖樣

Sandy Steen Bartholomew 是新罕布夏州 *Beez in the Belfry* 工作室的主理人，也是影響深遠的《Yoga for Your Brain》系列作者，貢獻了多個普及圖樣。

### 28. Zinging Static — Static 變體（見前述條目）

### 29. W2 — W2 結繩紋

- **創作者：** Sandy Steen Bartholomew
- **首見：**《Yoga for Your Brain》；後收錄於 TanglePatterns.com。
- **畫法：**
  1. 畫一直行相疊的「W」形。
  2. 旁邊再畫一直行 W 形，與前一行錯開半個 W，使之互鎖。
  3. 兩行交織。
  4. 中間夾出的小菱形填黑。
- **為什麼重要：** W2 用單一字母製造強烈的編織感。

### 30. Striping — 條紋紋 / 線陣紋

- **創作者：** Sandy Steen Bartholomew
- **首見：**《Yoga for Your Brain》。
- **畫法：**
  1. 在區塊內畫粗細與間距變化的平行線。
  2. 視需要讓線條跟隨區塊輪廓彎曲。
  3. 三到五條為一組，粗細交錯。
- **為什麼重要：** Striping 是最簡單的圖樣之一；作為冷靜填料，可平衡熱鬧的鄰居。

### 31. Zig zag（Zander） — 茲格紋

- **創作者：** Sandy Steen Bartholomew（Zander 是她兒子的名字）
- **首見：**《Yoga for Your Brain》。
- **畫法：**
  1. 在區塊上畫一條連續的鋸齒線——尖峰與尖谷交替。
  2. 旁邊再畫一條鋸齒線，與前一條鏡射，中間形成一排菱形。
  3. 加斜線或把菱形填黑。
- **為什麼重要：** Zig zag 是 Striping 的「動感版」對照。

### 32. Nipa — 棕櫚葉紋 / 椰葉紋

- **創作者：** Sandy Steen Bartholomew
- **首見：**《Yoga for Your Brain》。
- **畫法：**
  1. 畫一條彎曲的長莖。
  2. 從莖兩側射出一對對尖葉，像棕櫚葉。
  3. 每片葉是長水滴形；沿莖左右交錯。
  4. 把交替的葉片以斜線或實心黑填滿。
- **為什麼重要：** Nipa 帶來熱帶、蕨類般的長條質感，特別適合 Phicops 與 Zendala。

### 33. Verdigogh — 松針紋 / 翠針紋

- **創作者：** Sandy Steen Bartholomew
- **首見：**《Yoga for Your Brain》。
- **畫法：**
  1. 畫一條彎曲的莖。
  2. 從莖上射出一束束短直線——每束 5–7 根針。
  3. 每束從同一點放射開來。
  4. 沿莖錯開排列。
- **為什麼重要：** Verdigogh 是冬季或常綠主題紙磚的「松針代表」。

### 34. Tortuca — 龜殼紋

- **創作者：** Sandy Steen Bartholomew
- **首見：**《Yoga for Your Brain》。
- **畫法：**
  1. 畫一個六角形。
  2. 內部再畫一個略小的六角形。
  3. 將整片區塊以六角形蜂巢鑲嵌。
  4. 每個六角形內加上一個小十字、點、或弧——像龜殼上的紋路。
- **為什麼重要：** Tortuca 是蜂巢質感的基礎圖樣——也是台灣常見的傳統蛋糕模子上的紋樣。

## C 組 — Linda Farmer 圖樣

Linda Farmer 於 2010 年創立 TanglePatterns.com，是線上禪繞圖樣典藏的守護者，本身也創作數個圖樣。

### 35. Auras — 光環紋

- **創作者：** Linda Farmer（命名此技法；做法本身更早）
- **首見：** TanglePatterns.com；《The Book of Zentangle》。
- **畫法：** Auras 嚴格來說是個「技法」，不是單一圖樣。在任何形狀外側畫一條細的偏移輪廓線（光環）；再畫第二、第三道。光環可以等距、可以漸變。也可以「往內」畫，向中心收縮。
- **為什麼重要：** Auras 是禪繞詞彙中最萬用的工具——把單一線條變成「光暈」「回聲」的感覺。

### 36. Sparkle — 閃光紋 / 留白紋

- **創作者：** Linda Farmer（命名技法；廣泛使用）
- **首見：** TanglePatterns.com；《Joy of Zentangle》。
- **畫法：** Sparkle 是「在圖樣中刻意留下一條未斷白線」以暗示高光的技法。在描繪長形物（一片葉、一個捲、一道光環）時，提筆瞬間，跳過 1–2 mm 再續筆，留下的白線就是光點。
- **為什麼重要：** Sparkle 是最便宜、效果最強的立體感技巧——像茶湯表面那一道反光。

### 37. Arukas — 阿魯卡斯 / 雪花輻紋

- **創作者：** Mihoko Kurosaki（黑崎みほ子），透過 Linda Farmer 的網站普及
- **首見：** TanglePatterns.com。
- **畫法：**
  1. 中央畫一個小圓。
  2. 從圓向外射出偶數條（8、12、16）直線輻射。
  3. 每對相鄰輻射之間畫一個三角楔，輕微疊到下一個楔。
  4. 沿輻射方向往外加一圈又一圈更小的楔。
- **為什麼重要：** Arukas 是最受歡迎的進階圖樣之一——對稱、催眠般的視覺效果，畫起來特別有成就感。

### 38. Rixty — 點線交錯紋

- **創作者：** Linda Farmer（多由其網站歸屬）
- **首見：** TanglePatterns.com。
- **畫法：**
  1. 畫一格方格網。
  2. 每隔一格放一個實心圓點。
  3. 其餘格內畫一對交叉的對角線。
  4. 整片讀起來像「點」與「X」的棋盤。
- **為什麼重要：** Rixty 是乾淨幾何的填料，適合做背景。

### 39. Quib — 奎布紋

- **創作者：** Linda Farmer
- **首見：** TanglePatterns.com。
- **畫法：**
  1. 畫一條垂直線。
  2. 從它射出一系列細的向上 J 形筆劃，間距均勻。
  3. 旁邊再畫另一條垂直線與另一組 J 形筆劃，方向相反。
  4. 兩行視覺上互鎖。
- **為什麼重要：** Quib 給垂直節奏——比 Striping 更輕巧。

## D 組 — 其他 CZT 貢獻者圖樣

### 40. Amaze — 迷宮紋

- **創作者：** Carole Ohl，CZT
- **首見：** TanglePatterns.com；CZT 課程常見。
- **畫法：**
  1. 鉛筆畫一個方形。
  2. 從一個角開始，畫一條連續的線往內螺旋——每一步轉 90°，像沒有牆的迷宮。
  3. 線終結在方形中央。
  4. 在相鄰方形重複；交錯起點，使螺旋互鎖。
- **為什麼重要：** Amaze 是經典的「迷宮」圖樣，慢慢畫特別療癒。

### 41. Wud — 樹皮紋 / 木紋

- **創作者：** Karry Heun，CZT
- **首見：** TanglePatterns.com。
- **畫法：**
  1. 畫一排垂直波浪線，像層層木板的剖面。
  2. 在每塊木板內畫巢套的橢圓——像年輪木紋。
  3. 視需要把每塊木板的一邊加深。
- **為什麼重要：** Wud 是任何「木質、樹皮、紋理」效果的首選——特別適合台灣讀者熟悉的檜木質感聯想。

## 如何閱讀這部字典

每一條都是「起點」，不是「終點」。禪繞畫是個活的傳統。當你讀到「畫一排半月」時，請預期：

1. **變化是被鼓勵的。** 大多數圖樣都有官方變體（Cadent 至少有三種已記錄變體；Flux 有 Maria 版與 Rick 版）。你自己畫出來的小差異，不是偏離傳統，而是傳統的延續。
2. **順序很重要。** 禪繞教學強調「永遠用筆上墨，不要先用鉛筆打稿」（除了 string 與陰影外）。這逼你做出承諾、相信過程。上文每一個步驟，都對應一筆或一小組筆劃；一組與一組之間請暫停一下。
3. **陰影最後上。** 所有線條完成後，再用 2B 鉛筆與 tortillon 紙筆做陰影。
4. **失誤化為亮點。** 線條走偏了不擦掉——它變成新圖樣的種子。這是禪繞最深的一堂課，也很像台灣茶藝裡「殘缺也是美」的哲學。

一張典型的學習磚會用三到六個圖樣，每個填入 string 切出的不同區塊。建議初學者先精熟 Crescent Moon、Hollibaugh、Printemps、Tipple、Florz、Knightsbridge 這六個——光是這六個就能組合出數百張漂亮的紙磚。

## 進一步學習資源

- **TanglePatterns.com** — Linda Farmer 的典藏網站：https://tanglepatterns.com 是線上最權威的免費資源。截至 2026 年，已收錄超過 1,800 個圖樣，每個都有文字步驟，多數附有歸屬說明。「Tangle Patterns String Theory」專區也保存了每日練習用的 string。
- **《The Zentangle Primer》第 1、2 卷**（Rick Roberts & Maria Thomas）—— 創辦人本人撰寫的官方參考。
- **《One Zentangle a Day》**（Beckah Krahula）—— 6 週每日課程；對新練習者極佳。
- **《Yoga for Your Brain》及續作**（Sandy Steen Bartholomew）—— 親切、有趣的入門。
- **《Joy of Zentangle》**（Sandy Bartholomew、Suzanne McNeill、Marie Browning）—— 廣泛收錄多位 CZT 的貢獻。
- **ZenAgain** —— 禪繞官方年度大會；每年都有新圖樣首次公開。

這部字典是入口。完整的禪繞練習是一條長路，每一張紙磚都會悄悄為你的個人詞彙加入新變體。
