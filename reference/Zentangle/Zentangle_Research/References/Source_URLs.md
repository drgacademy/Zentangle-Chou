## Adding New URLs

1. Verify the URL loads without redirect loops or paywalls.
2. Capture the page title verbatim (use it as the link text).
3. Add the entry to the most appropriate section above.
4. Append `— accessed YYYY-MM-DD` using the day you verified it.
5. If the URL replaces an outdated source, mark the old entry as superseded — do not delete.

## Removing URLs

Never delete. Mark as:
- `(404 as of YYYY-MM-DD)` if dead.
- `(behind paywall as of YYYY-MM-DD)` if access was lost.
- `(superseded by [new URL])` if the resource moved.

This preserves citation chains for everything we publish.
